- [x] 1. 定位 luxury-terminal-list 对应组件与样式
**关键解决方法**：根因是 flex 布局未给路由视图/主内容区正确的高度约束（缺少 min-h-0/h-full/flex 链路），导致侧边栏列表被父容器裁剪但自身不产生 overflow。通过在 App.vue 的 main+router-view 组件链路和 HomeView 关键容器补齐 min-h-0/h-full，并对列表容器隐藏滚动条（但保留滚动）解决。
**不能犯的错误**：只在子元素加 overflow-y-auto 不够：父级 flex item 默认 min-height:auto 会阻止滚动区域收缩，从而出现“看不到下面但也滚不动”的裁剪问题。
